# React Ask App

# DEVELOPMENT

## Instruction

1. `$ npm install`

2. `$ npm run dev`

3. connect to [http://localhost:8080](http://localhost:8080)

## Directory Structure

```
client/
  * public assets (images, fonts, css files)
server/
  api/
  render/
  * express + mongoose
src/
  * reactjs + flux
```
